# TaskManager MVP

SwiftUI MVP для личного приложения под iPhone.

## Что внутри

- Финансы: периоды выплат, доходы, постоянные и разовые расходы, заметка по будущим тратам.
- Ежедневник: недельный экран, задачи по дням, разделы `Личное`, `Мантера`, `Красная Поляна`, галочки выполнения.
- Продукты: категории продуктов из таблицы, отметки покупок, редактирование списка.
- Блюда: список блюд с продуктами, рецептом и КБЖУ.

Данные хранятся локально на устройстве в JSON-файле приложения.

## Запуск на iPhone

1. Установите Xcode из Mac App Store.
2. Откройте `TaskManager.xcodeproj`.
3. Подключите iPhone кабелем и разблокируйте его.
4. В Xcode сверху выберите устройство вместо симулятора.
5. Откройте target `TaskManager` -> `Signing & Capabilities`.
6. В `Team` выберите свой Apple ID. Если его нет: `Xcode` -> `Settings` -> `Accounts` -> добавьте Apple ID.
7. Если Xcode попросит уникальный Bundle Identifier, замените `com.local.TaskManagerMVP` на что-то свое, например `com.yourname.TaskManagerMVP`.
8. Нажмите Run.
9. Если iPhone попросит доверять разработчику: `Settings` -> `General` -> `VPN & Device Management` -> выберите свой Apple ID -> `Trust`.

Для бесплатного Apple ID приложение обычно живет на телефоне ограниченное время, после этого его нужно снова запустить из Xcode.
